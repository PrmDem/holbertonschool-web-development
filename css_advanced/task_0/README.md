# CSS Advanced

This project focuses on using CSS to turn a design into a real web page. CSS is used to control layout, spacing, colors, typography, and alignment so the final page matches the intended design.<br/>Without it, our page wouldn't be very agreeable to visit, nor even easy to navigate. Design choices can also make a huge difference in terms of accessibility, which must always be taken into consideration while building a web page.<br/>
