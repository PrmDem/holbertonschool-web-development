# Advanced HTML

Web development has changed a lot since its inception – as did everything else. A marked difference between then and now is the use of tools like Figma or Balsamiq to create wireframes or design prototypes. This project is meant to teach us how to __implement a webpage from a Figma design file__, as it is widely used in front-end development — therefore a must-know for any future endeavours.<br/>
